Learning jQuery

July 15, 2014

So you want to learn jQuery? Well first, you need a solid foundation in JavaScript. Once you got that down, come back here and I'll teach you jQuery in this article.

So what is jQuery? jQuery is a fast, small, and feature-rich JavaScript library. It makes things like HTML document traversal and manipulation, event handling, animation, and Ajax much simpler with an easy-to-use API that works across a multitude of browsers. With a combination of versatility and extensibility, jQuery has changed the way that millions of people write JavaScript.

jQuery is a global variable and a shorthand form of it is $. Inside the parameters of jQuery function, you put in DOM elements. DOM stands for Document Object Model. Example of a DOM element are p, ul, body, etc. It is basically html tags. Once you select a DOM element, you can apply an method to it.
