About

"Programming" is thinking, not typing. Most of programming is spent sleeping, walking around, staring out the window, or doing anything else that helps you relax and think. Code can be one of the best tools available, but only when the design of the code you’re consuming has been considered. Writing great code is an art — finding the perfect balance between simplicity and functionality, opinions and open-mindedness. Functional programming and object orientation are merely a reflection of each other, a form of computational yin and yang.
